Welcome to Programming for Economists¶

This is the webpage for the BSc course Programming for Economists at Tilburg University. This course is taught in the first year for ECO and EBE.

Note that this webpage is under construction for 2017/8

Team¶

In 2017-2018 this course is taught by:

  • Santiago Bohorquez Correa
  • Jan Boone
  • Clemens Fiedler
  • Mark Klik
  • Misja Mikkers
  • Minke Remmerswaal

Make sure that you read the Rules of the Game if you want to attend this course!

Datacamp¶

We are very happy that we partner with datacamp for this course to teach you both python and R.

Datacamp offers great on-line courses for you to learn R and python and it allows us to track your progress...

Important¶

Things to do if you want to follow this course:

  • enroll in datacamp with your Tilburg University email address
  • sign up at github.com
  • create a repository called “assignments” as explained here; and save the address of this repository
  • after you have done this, go to the webform:
  • and fill in this webform.
  • note that you need to fill in the webform before March 15, 2017
  • if you do not fill in the webform before the deadline, you cannot get a grade for this course
