Welcome to Programming for Economists¶

This is the webpage for the BSc course Programming for Economists at Tilburg University. This course is taught in the first year for ECO and EBE.

Note that this webpage is under construction; lectures are not complete yet and some links to screencasts do not work

Team¶

In 2016-2017 this course is taught by:

  • Santiago Bohorquez Correa
  • Jan Boone

Make sure that you read the Rules of the Game if you want to attend this course!

Datacamp¶

We are very happy that we partner with datacamp for this course to teach you both python and R.

Datacamp offers great on line courses for you to learn R and python and it allows us to track your progress...

Important¶

Things to do if you want to follow this course:
